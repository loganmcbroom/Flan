Detailed Description

Graph is a general graphing object. This class stores image data in a std::shared_ptr.

Member Enumeration Documentation

◆ WaveformMode

enum class flan::Graph::WaveformMode
strong

This is used by waveform graphing functions. Direct displays any sampled function data as-is. Symmetric uses sample data magnitudes and displays the sampled data symmetrically around the waveform center line.
