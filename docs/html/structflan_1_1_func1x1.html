Generate an ADSR envelope. Generated envelopes always range from 0 to 1. The Exp parameters dictate how the envelope curves between fixed points. Using an Exp of 1 gives a linear movement. Values between 0 and 1 give curves that move rapidly toward their destination before slowing down. Values larger than 1 do the opposite. Other values will give unusual, but well defined, behaviour, so no clamping is applied.
