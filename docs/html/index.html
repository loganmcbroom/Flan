Flan - A general purpose non-real-time audio processing library

The Flan library contains buffer classes for data representing sound along with algorithms for creative manipulation of those data types. Additional utilities for working with sound in a creative capacity such as wavetable synthesis algorithms are also included. A high degree of technical knowledge of sound is not required, but an understanding of short-time Forier transforms will be useful in handling PVOC data.
